Question 1 :  A quoi sert le fichier "package.json" créé par la commande npm init?
-- Un fichier de configuration obligatoire
-- Pour rentrer un peu plus en détails, le package. json est véritablement le coeur de tout projet utilisant node. 

Question n°2 : A quoi sert le fichier "package-lock.json" généré suite à l’installation d'un package npm ?
-- Ce fichier sert a repertorier tout les extention de node avec toute les info dessus (ex : version de l'extention)
